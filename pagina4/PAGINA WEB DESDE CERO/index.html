CSS GRID

En CSS es fundamental conocer la sintaxis básica (selectores, propiedades y valores), el modelo de caja (box model), cómo funciona el flujo del documento y posicionamiento (static, relative, absolute, fixed), el sistema de diseño con Flexbox y Grid, el uso de media queries para crear diseños responsivos, las pseudo-clases y pseudo-elementos (:hover, ::before, etc.), la especificidad y cascada (cómo se aplican los estilos), el uso de variables CSS (custom properties), la creación de animaciones y transiciones, así como buenas prácticas para mantener el código limpio y reutilizable.
